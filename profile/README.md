## 📦 Our Projects

### 🛍️ **Lootifyr** - E-Commerce Distribution Platform
- **Description:** A modern solution for streamlining offer distribution across multiple e-commerce platforms.
- **Tech Stack:** TypeScript
- **Purpose:** Enables sellers, affiliates, and automated systems to dispatch promotional offers from a unified interface or API

### 🤖 **Jericho Discord Bot** - Community Management Made Easy
- **Description:** A powerful, customizable Discord bot designed to make servers smarter, smoother, and more fun
- **Tech Stack:** TypeScript
- **Features:** Fast responses, utilities, community management, and entertainment
- **Bot Invite:** [Jericho](https://top.gg/bot/1470079725106888817)

### 🔐 **AuthCore** - Minecraft Fabric Authentication Framework
- **Description:** A lightweight, server-side authentication framework for Fabric servers
- **Tech Stack:** Java
- **Features:** Secure player registration and login workflows for offline-mode servers with modular, configurable design
- **Repository:** [AuthCore](https://github.com/PotenFYR-Studios/AuthCore) 
- **Modrinth:** [AuthCore Modrinth](https://modrinth.com/mod/authcore)

### 📊 **StatFYR** - Minecraft Statistics Plugin
- **Description:** A statistics tracking system for Minecraft servers
- **Tech Stack:** Java
- **Repository:** [statfyr](https://github.com/PotenFYR-Studios/statfyr)
- **Modrinth:** [Statfyr Modrinth](https://modrinth.com/plugin/statfyr)

### 🎮 **Minecraft Plugins**
- **OneJumpAllJump** - A chaotic Spigot plugin where a single jump launches the entire server! 🚀
- **EchoingDeaths** - Adds immersive death-based curses to nearby players with temporary debuffs themed around causes of death.
